# Luxury Automotive Site

React/Vite site for The Automotive Group brand pages.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Assets

Source images are stored in `pics/`. Optimized web assets are generated into
`src/assets/optimized/`.

```bash
npm run optimize-images
```

## Pages

- `/` - brand overview
- `/brands/zeigler-bailey`
- `/brands/encor`
- `/contact`
